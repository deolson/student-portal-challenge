# Student Portal

## Description
A college portal to enter student first name, last name, DOB, address, and a profile photo.

## Features

Users can:
* Login/Logout.
* Modify their information.
* Upload a profile picture.
* View his own profile.

## Technologies Used

* React
* Redux
* TypeScript
* Spring Data JPA
* Spring Boot
* Docker
* EC2
* S3
* Java11
